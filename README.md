# Digital Beedi Branch Management System

A complete digital solution to replace traditional manual processes in the beedi industry.

# 📌 Project Overview

The Digital Beedi Branch Management System is designed to modernize and automate day-to-day operations in beedi production units. This system digitizes attendance tracking, production counting, wage management, and inventory monitoring to increase efficiency, transparency, and accuracy. It provides a user-friendly interface suitable for workers and contractors with minimal digital literacy.

# 🎯 Key Features

✅ 1. Digital Attendance Tracking

->Automates daily attendance recording

->Eliminates manual paper registers

->Stores attendance securely for wage calculation

✅ 2. Production & Wage Management

->Tracks daily beedi production per worker

->Calculates earnings automatically based on production

->Ensures fair and transparent wage distribution

->Supports UPI / digital payment integration

✅ 3. Inventory Tracking

->Monitors raw materials and finished goods

->Reduces wastage and improves stock accuracy

->Provides real-time updates to contractors

✅ 4. User-Friendly Interface

->Simple UI designed for workers with low digital literacy

->Clear navigation, large buttons, and easy workflows

✅ 5. Real-Time Reporting

->Daily, weekly, and monthly productivity reports

->Financial summaries for contractors

->Worker-wise and category-wise insights

# 🛠️ Challenges & Solutions

Challenge:

Most beedi workers and contractors have very limited digital literacy, making it difficult to adopt software-based workflows.

Solution:

We developed a simple, intuitive, icon-based interface that works on basic smartphones. The system minimizes text input and supports step-by-step guided navigation.

# 📊 Expected Outcome

->Faster and more accurate daily operations

->Reduction in manual errors

->Increased transparency in wages

->Efficient inventory management

->Empowerment of workers through a digital platform
